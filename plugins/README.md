# Plugins

Curated plugins from community repositories for Power BI and Microsoft Fabric development. Each plugin is vendored into `plugins/` as a self-contained package, and source attribution is recorded here.

## Plugins

| Plugin | Folder | Source | Notes |
|---|---|---|---|
| fabric-skills | [`fabric-skills/`](fabric-skills/) | `https://github.com/microsoft/skills-for-fabric/tree/main/plugins/fabric-skills` | Microsoft Fabric skill plugin from the Skills for Fabric repo. |
| powerbi | [`powerbi/`](powerbi/) | `https://github.com/RuiRomano/powerbi-agentic-plugins/tree/main/plugins/powerbi` | Power BI development plugin from Rui Romano's Power BI agentic plugins repo. |
| fabric | [`fabric/`](fabric/) | `https://github.com/RuiRomano/powerbi-agentic-plugins/tree/main/plugins/fabric` | Fabric platform plugin from Rui Romano's Power BI agentic plugins repo. |
| reports | [`reports/`](reports/) | `https://github.com/data-goblin/power-bi-agentic-development/tree/main/plugins/reports` | Power BI report development plugin from data-goblin's repo. |
| azure-skills | [`azure-skills/`](azure-skills/) | `https://github.com/microsoft/skills` | Azure/Entra/Foundry skill plugin from Microsoft's skills repo. |
| deep-wiki | [`deep-wiki/`](deep-wiki/) | `https://github.com/microsoft/skills` | Repository documentation/wiki-generation plugin from Microsoft's skills repo. |
